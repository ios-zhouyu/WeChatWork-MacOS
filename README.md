## 功能

* 消息防撤回
* 会话去水印



## 安装

```bash
git clone https://github.com/ivothgle/WeChatWork-MacOS.git && cd WeChatWork-MacOS && ./wewtool.sh install
```

卸载

```bash
./wewtool.sh uninstall
```
